A single feedback-post screen: a vote tally, title and metadata, a Markdown
description, a comment thread (avatar + author + relative timestamp), and a
comment composer. Use as a template for an idea / discussion detail view.
Composed from `Markdown`, `Avatar`, `UserName`, `Moment`, `Form`, `TextArea`, and
`Button`. Takes no props — render `<PostDetail />`.
